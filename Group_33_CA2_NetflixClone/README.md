# DevSecOps Netflix Clone

### Problem
Many CI/CD pipelines lack integrated DevSecOps security and monitoring.
(https://stackoverflow.com/questions/73490952/devsecops-q-how-do-you-build-in-the-capability-to-automate-the-ability-to-disco)

### Solution
A secure Netflix Clone deployed using Jenkins, Docker, Kubernetes, SonarQube, Trivy, Prometheus, and Grafana.
(https://github.com/<yourusername>/DevOps-CA2)

### Tools Used
Jenkins, Docker, Kubernetes (Minikube), Ansible, Prometheus, Grafana, SonarQube, Trivy.
